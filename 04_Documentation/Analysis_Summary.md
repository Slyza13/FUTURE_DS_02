# RavenStack Retention Analysis - Key Findings

## Project Overview
Analyzed 500 SaaS customer accounts across 5 relational tables to identify churn patterns 
and retention drivers for a fictional AI-tools startup.

## Key Metrics
- Total Customers Analyzed: 500
- Churn Rate: See Executive Summary output
- Average Customer Lifetime: See analysis notebook

## Top Insights
1. **Support-Driven Churn:** Higher ticket volumes correlate strongly with churn
2. **Engagement Matters:** Low feature adoption in first 14 days predicts churn
3. **Beta Feature Risk:** Beta users experience more errors and higher churn
